Manual & Instruktioner:

  • 📄 Kopiera: Klicka på ikonen för att kopiera infon till urklipp.
  • ✏️ Ändra: Klicka på pennan för att redigera en sparad kalkyl.
  • 💰 Kalkylberäkning: Fyll i ursprunglig och korrigerad summa, systemet räknar ut mellanskillnaden.
  • ⚡ Snabbval: Använd knapparna för att snabb-utföra kommentarer för vanliga kalkylfel.
  • 📸 Bilder: Du kan dra-och-släppa bilder i rutan eller klistra in direkt med Ctrl+V.
  • 📂 Excel-export: Skapar en detaljerad CSV-fil med all data för Excel.
  • 🤝 Slå ihop: Kombinera dina kontroller med filer från kollegor. Vid import väljer du om det är din egen backup (förblir blå) eller från en kollega (blir lila).
  • 💾 Backup: Spara en .json-fil för att säkra din data.
  • 📈 Statistik: Klicka på "Öppna Statistikvy" för djupanalys av fel och summor.
  • 📋 PDF-Rapport: Spara till PDF spara ner via statistikvyn (💡Skapa den i LIGHT MODE).
  • ❌ Rensa: Använd "Rensa minne" för att tömma hela listan.
  • 💾 Backup-påminnelse: En röd varning visas vid backup-knappen om det gått över 48 timmar sedan sist och du har gjort nya ändringar.
  • 🔍 Dubbelkontroll: Registreringsnummer kontrolleras automatiskt; fältet blir gult om bilen redan finns i listan för att undvika dubbelarbete.
  • 🏢 Verkstad: Sök i listan eller skriv fritt; systemet är sökbart och minns de 10 senaste verkstäderna du använt för att snabba på inmatningen.
  • ⚡ Snabbinmatning: Klicka på blixten vid verkstadsfältet för att fylla i samma verkstad och skapare som den förra kalkylen du sparade.
  • 🚩 Flagga för uppföljning: Klicka på 🚩-ikonen på en rad för att flagga kalkylen. Du kan lägga till en kommentar, t.ex. "Inväntar svar från verkstad". Håll muspekaren över flaggan för att se kommentaren.
  • 🔍 Besiktning: Markera en kalkyl för besiktning vid skapandet med kryssrutan, eller klicka på 🔍-ikonen på raden i efterhand. Besiktningsrader visas i ljusblått. Klicka på "Visa besiktning" för att filtrera fram dessa.
